What is a Markov Chain?

A Markov chain is a mathematical model that describes a sequence of events where the probability of each event depends only on the state attained in the previous event. This "memoryless" property makes them incredibly useful for modeling systems where the current state contains all the information needed to predict the future.

Mathematical Foundation

Formally, a Markov chain is defined by:

  • State Space (S): The set of all possible states
  • Transition Matrix (P): A matrix where P(i,j) represents the probability of transitioning from state i to state j
  • Initial Distribution: The probability distribution over the initial states

The key property is the Markov property:

P(X_{n+1} = j | X_n = i, X_{n-1} = i_{n-1}, ..., X_0 = i_0) = P(X_{n+1} = j | X_n = i)
